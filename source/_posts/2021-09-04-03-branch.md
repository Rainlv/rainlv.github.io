## 创建
    git checkout -b <分支名>  // 创建并切换到分支
    
    git branch <分支名>  // 创建分支
    
    git checkout <分支名> // 切换分支

## 查看

    git branch // 查看分支，列出所有分支

## 合并

    git merge <分支名>  // 合并分支到当前分支

## 删除

    git branch -d <分支名>  // 删除分支
